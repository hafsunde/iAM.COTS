# iAM.COTS

<!-- badges: start -->

<!-- badges: end -->

iAMCOTS implements an extended Children-of-Twins (COTS) model with
indirect assortative mating, spouses, and two offspring per nuclear
family. The model is estimated in OpenMx and supports MZ, DZ,
full-sibling, half-sibling, and unknown-zygosity family structures.
